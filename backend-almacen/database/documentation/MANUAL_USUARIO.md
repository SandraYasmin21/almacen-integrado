# Manual De Usuario

## Inicio De Sesion
1. Ingresar usuario y contrasena.
2. El sistema valida que el usuario este activo.
3. El menu visible depende del rol asignado.

## Inventario
1. Registrar articulos desde Catalogo Central.
2. Usar Recepcion de Mercancia para alta individual o importacion Excel/CSV.
3. Consultar existencias en Almacen Fisico.
4. Usar Historial y Bitacora para revisar movimientos.

## Movimientos
1. Registrar entrada, salida, prestamo o devolucion.
2. Para trazabilidad avanzada usar transferencia, envio a reparacion, retorno, cambio de responsable o baja logica.
3. Todo movimiento genera folio, usuario, fecha y bitacora.

## Vehiculos
1. Registrar vehiculos desde Flotilla Vehicular.
2. Capturar mantenimientos preventivos/correctivos y lecturas de kilometraje.
3. Registrar gastos extra.
4. Consultar dashboard, kilometraje e historial por vehiculo.

## Proyectos
1. Crear proyecto con cliente, responsable y estatus.
2. Asociar recursos desde la API de proyectos.
3. Consultar reportes de activos por proyecto.

## Reportes
1. Abrir Reportes Basicos.
2. Seleccionar el reporte requerido.
3. Revisar resultados por estado, ubicacion, responsable, proyecto, categoria, materiales, vehiculos o movimientos.
